The Artifact

Travlr Getaways is a travel booking application originally built in CS 465 (Full Stack Development). The app uses the MEAN stack — MongoDB, Express, Angular, and Node.js — with a customer-facing website, an Angular admin SPA, and a REST API with JWT authentication.

Using one artifact across all three enhancement categories demonstrates how different computer science disciplines interconnect in real-world applications. Testing infrastructure from Enhancement One supported validation of the search optimizations in Enhancement Two, while both informed the security implementations in Enhancement Three.

Additional Improvements

Beyond the three core enhancements, nine improvement branches were developed, reviewed, and merged:

  • Controller Modernization — Refactored 7 controllers from the deprecated request module to native fetch with async/await
  • Direct Database Queries — Replaced inefficient HTTP self-calls in the customer app with direct Mongoose queries
  • Scalable Pagination — Added pagination to trips and bookings APIs for both small datasets and large catalogs
  • Data Model Standardization — Converted pricing fields from strings to proper numeric types to prevent calculation bugs
